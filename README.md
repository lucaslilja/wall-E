# Requirements
You need to have Java 21 installed

# Testing
To test the project, run the following command:

```mvn clean test```

# Running the project
To run the project, use the following command:

```
mvn clean package
java -jar ./target/walle-0.1.0-SNAPSHOT.jar
```
